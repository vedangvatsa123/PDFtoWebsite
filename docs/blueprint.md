# **App Name**: ResumeRack

## Core Features:

- Resume Upload: Allow users to upload their resume in common formats (e.g., PDF, DOCX).
- Profile Generation: Automatically extract key information from the resume to pre-populate a user profile.
- Profile Editor: Provide a user-friendly interface for editing and refining the extracted information.
- Customization: Enable users to select from a range of visually appealing, professional themes to style their profile page.
- Page Publishing: Allow users to publish their profile as a unique, shareable web page.
- Hosting: Hosting each page as server-side-rendered react page in order to improve SEO.
- Dark Mode: Allow users to switch to a dark mode for comfortable viewing in low-light conditions.

## Style Guidelines:

- Primary color: Light blue (#E3F2FD) to represent knowledge and serenity.
- Secondary color: White (#FFFFFF) for a clean and modern look.
- Dark Mode Background: Dark gray (#333333) for comfortable viewing in low light.
- Dark Mode Text: Light gray (#DDDDDD) for readability in dark mode.
- Body font: 'Roboto' sans-serif for clear and readable text.
- Headline font: 'Montserrat' sans-serif for modern, eye-catching headings.
- Use simple, line-based icons to represent different sections of the resume (e.g., work experience, education, skills).
- Employ a clean, single-column layout with ample white space to ensure readability and focus.
- Subtle transitions and animations (e.g., fade-in effects) when navigating between profile sections or saving changes.
